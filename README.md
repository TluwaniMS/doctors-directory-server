# doctors-directory-server

# About Repository:
This is a basic node-js project that I created to learn how to connect node-js to an SQL database(Postrgres)

# Software(s) required:
* Latest stable version of Node Package Manager
* Postgres Database

# Dependencies:
* CORS
* DOTENV
* EXPRESS
* BCRYPT
* JSONWEBTOKEN
* MORGAN
* PG
* PG-HSTORE
* SEQUELIZE
* HELMET

# Dev-dependencies:
* NODEMON
* PRETTIER
